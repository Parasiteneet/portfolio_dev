## About EZBKING

個人でも予約を受けられるWebアプリケーションです。予約の登録・変更・削除・を行えます。誰でも簡単に使用することが可能で、面倒な登録をしなくても利用できます。

## 使用技術
- PHP 7.4.15
- Laravel 5.8
- MySql 5.7.34
- Apache 2.4.46
- AWS
  - VPC
  - EC2
  - Route53
- PHPUnit
- Vue.js

## 機能一覧

- ユーザー登録(Auth)
- ログイン機能(Auth)
- 予約機能
  - 新規予約
  - 予約の変更
  - 予約の削除
  - 予約日時の確認
- メール機能(Mailable)
  - 予約時に管理人のメールアドレスにメッセージが届く。
